## 模块汇总
- 1.主控芯片（STM32F103C8T6）：处理多传感器数据
- 2.WIFI模块（ESP8266）：上传传感器数据到app，接收app数据             		通信方式：UART通信/MQTT协议
- 3.显示模块（OLED 0.96寸4针脚）：OLED屏幕显示数据							          通信方式：I2C通信
- 4.温湿度监测（DHT11）：精准测量温湿度									                  通信方式；1-Wire单总线协议
- 5.开关盖检测（红外对管TCRT5000）：检测药箱盖是否打开						         通信方式：ADC模拟信号采集
- 6.药品重量检测（HX711称重模块）:检测药品重量								            通信方式：2线串行（时钟+数据）
- 7.定时提醒（蜂鸣器PA7）：用户预设的吃药时间，通过蜂鸣器声音提醒吃药				   通信方式：GPIO控制
- 8.时钟模块（DS1302）：为单片机提供时间数据								              通信方式：3线串行（时钟+使能+数据）
- 9.出药装置（舵机SG90）：使用舵机模拟出药									                  通信方式：PWM控制
- 10.语音播报（SYN6288 语音合成模块+喇叭）：TTS文本转语音,提醒吃药						通信方式：UART通信
- 11.供电模块（AMS1117线性稳压器）：系统提供电源,同时保证5v稳压输出。

### 数据发送流程
在freertos.c文件的ESP_SendTask任务中实现了MQTT数据发送：

1. 传感器数据采集（温湿度、重量、红外传感器等）
2. 数据格式化为JSON字符串
3. 通过MQTT协议发布到指定主题

```c
void ESP_SendTask(void *argument)
{
    char josnstr[200];          /* 即将发送的json字符串 */
    while(1)
    {
        // 格式化JSON数据
        sprintf(josnstr, "{\"code\":\"%s\",\"temp\":%d,\"humi\":%d,\"beep\":%d,\"sg90\":%d,\"tcrt5000\":%d,\"weight\":%.1f}",
                MQTT_TOPIC, DHT11_Data.temp, DHT11_Data.humi, BEEP, SG90, TCRT5000, Weight);
        
        // 发布MQTT消息
        MQTT_PublishData(josnstr, 0);
        vTaskDelay(1000);  // 每秒发送一次数据
    }
}
```

#### 命令接收流程
在freertos.c文件的ESP_ReceiveTask任务中实现了MQTT命令接收与处理：

1. 监听MQTT订阅的命令主题
2. 解析接收到的控制命令
3. 执行相应的操作（蜂鸣器开关、舵机控制等）

```c
void ESP_ReceiveTask(void *argument)
{
    while(1)
    {
        if(Usart2_RxBuff[0] != 0x00){
            // 处理蜂鸣器控制命令
            if(strstr(Usart2_RxBuff,"beep:on")){ BEEP=1; BEEP_OUT(0); }
            if(strstr(Usart2_RxBuff,"beep:off")){ BEEP=0; BEEP_OUT(1); }
            
            // 处理舵机控制命令
            if(strstr(Usart2_RxBuff,"sg90:on")){
                SG90=1; set_servo_angle(90); BEEP=1; BEEP_OUT(0);
                // SYN_FrameInfo(0,"吃药时间到了，请吃药");  // 语音提醒
            }
            if(strstr(Usart2_RxBuff,"sg90:off")){ SG90=0; set_servo_angle(0); BEEP=0; BEEP_OUT(1); }
            
            // 处理重量提醒命令
            if(strstr(Usart2_RxBuff,"weight:on")){
                BEEP=1; BEEP_OUT(0);
                SYN_FrameInfo(0,"药品余量不足，请及时加药");  // 语音提醒
            }
            
            // 清空接收缓冲区
            Usart2_RxCounter = 0;
            memset(Usart2_RxBuff, 0, USART2_RXBUFF_SIZE);
        }
        vTaskDelay(300);  // 每300ms检查一次
    }
}
```

## 通信架构 ：
- 移动端应用与嵌入式系统通过MQTT协议进行通信
- 嵌入式系统通过ESP8266连接WiFi网络接入MQTT服务器
- 支持双向通信：移动端发送控制命令，嵌入式系统上报传感器数据
- 数据流程 ：
  - 1. 传感器采集环境数据（温湿度、重量等）
  - 2. 数据通过MQTT上传到服务器
  - 3. 移动端应用订阅MQTT主题获取实时数据
  - 4. 移动端可发送控制命令（如开关药箱、设置阈值）
  - 5. 嵌入式系统执行命令并反馈结果
